# JLU Templates

吉林大学作业及论文模板，基于现代排版系统 Typst 开发。

## 项目简介

这是一个为吉林大学学生设计的 Typst 模板集合，提供了作业、论文等多种文档类型的排版模板，帮助学生快速生成符合学校规范的文档。

## 功能特点

- **多文档类型支持**：包含本科毕业论文、思政作业等多种模板
- **规范排版**：遵循吉林大学相关文档格式要求
- **灵活配置**：可自定义字体、页面布局等参数
- **多语言支持**：内置中文、英文、韩文等语言设置
- **便捷易用**：通过简单的参数配置即可生成专业文档

## 使用方法

### 前置条件

1. 安装 Typst：请参考 [Typst 官方文档](https://typst.app/docs/) 进行安装
2. 了解 Typst 基本语法：可通过官方教程快速入门

### 快速开始

1. 克隆或下载本仓库
2. 根据需要选择相应的模板文件进行编辑
3. 使用 Typst 编译生成 PDF 文档

### 模板使用示例

#### 本科毕业论文

```typst
#import "页面/本科毕业论文封面.typ": 封面
#import "页面/摘要及关键词.typ": 摘要
#import "页面/目录.typ": 目录
#import "配置/页面.typ": *
#import "配置/语言.typ": 基本英文
#import "lib.typ": *

#show: 应用配置.with(配置: (段落, 边距, 默认字体))

#封面(
  中文题目: ("你的论文题目",),
  外语题目: ("Your Thesis Title",),
  学生姓名: "你的姓名",
  学号: "你的学号",
  日期: "2024年6月",
  专业: "你的专业",
  学院: "你的学院",
  指导教师: "指导教师姓名",
  职称: "教授",
)

// 添加摘要及关键词
#摘要(
  摘要: [你的摘要内容],
  "关键词1", "关键词2", "关键词3"
)

// 添加英文摘要
#show:基本英文
#摘要(摘要: [Your abstract content], "keyword1", "keyword2", "keyword3")

// 添加目录
#目录

// 开始正文
= 绪论
== 课题研究背景及意义
// 你的正文内容
```

### 自定义配置

你可以在 `配置/` 目录下的文件中自定义字体、页面布局等参数：

- **字体配置**：在 `字体.typ` 中修改字体设置
- **页面配置**：在 `页面.typ` 中调整页边距、段落格式等
- **语言配置**：在 `语言.typ` 中设置多语言支持
